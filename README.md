# hash_cracker
A multi-algorithm hash cracking utility built for security auditing. This tool uses a wordlist-based approach to decrypt common cryptographic hashes, assisting in password strength assessments.

# Key Features:

Supports MD5 and SHA-256 hashing algorithms.

Dynamic wordlist input for large-scale cracking operations.

Optimized string comparison for efficient processing.
